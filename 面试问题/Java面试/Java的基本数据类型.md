Java 的基本数据类型共有 **8 种**，它们不属于对象，直接存储在**栈（Stack）**中，因此性能更高。我们可以根据它们的特性分为四大类：

#### 1. **整数类型**

| **类型**    | **占用空间 (位/Byte)** | **取值范围 (约)**    | **默认值** |
| --------- | ----------------- | --------------- | ------- |
| **byte**  | 8-bit (1B)        | -128 到 127      | 0       |
| **short** | 16-bit (2B)       | -3.2万 到 3.2万    | 0       |
| **int**   | 32-bit (4B)       | -21亿 到 21亿 (常用) | 0       |
| **long**  | 64-bit (8B)       | 非常大 (需加 L 后缀)   | 0L      |

#### 2.浮点类型

|**类型**|**占用空间 (位/Byte)**|**说明**|**默认值**|
|---|---|---|---|
|**float**|32-bit (4B)|单精度 (需加 f 后缀)|0.0f|
|**double**|64-bit (8B)|双精度 (常用)|0.0d|

#### 3. 字符类型

|**类型**|**占用空间 (位/Byte)**|**说明**|**默认值**|
|---|---|---|---|
|**char**|16-bit (2B)|采用 Unicode 编码|'\u0000' (空)|

#### 4.布尔类型

|**类型**|**占用空间**|**取值**|**默认值**|
|---|---|---|---|
|**boolean**|理论 1-bit|true / false|false|
